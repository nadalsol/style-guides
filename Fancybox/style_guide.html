Flexible images

Images

Muy importante! Dejar los atributos width y height del tag img vacíos.

1. Para hacer que la imagen sea flexible (que se adapte al tamaño de su contenedor), hay que encapsular-la dentro de un contenedor de bloque (p.e. un <span> con display:block).

→ Mostrar código 

  • <span class="flxImg">
    <img src="img/themes/woman/bg_stage.jpg" alt=" " />
</span>

2. Si necesitamos que la imagen sea un enlace podemos sustituir el tag <span> por un tag <a>, para que la imagen se comporte como un enlace sin dejar de ser felxible.

→ Mostrar código 

  • <a href="#" class="flxImg">
    <img src="img/themes/woman/bg_stage.jpg" alt=" " />
</a>

3. Opcionalmente se puede redefinir el ancho del contenedor en porcentaje (%), mediante una class (p.e. wp50). El valor del porcentaje determinará su ancho en función de su padre.

→ Mostrar código 

  • <span class="flxImg wp50">
    <img src="img/themes/woman/bg_stage.jpg" alt=" " />
</span>

4. Opcionalmente el contenedor puede ir flotado, mediante class floatL/floatR.

→ Mostrar código 

  • <span class="flxImg wp50 floatL">
    <img src="img/themes/woman/bg_stage.jpg" alt=" " />
</span>

5. Opcionalmente se puede definir un skin a la imagen, mediante una class skinFlxImg.

→ Mostrar código 

  • <span class="flxImg skinFlxImg">
    <img src="img/themes/woman/bg_stage.jpg" alt=" " />
</span>

Videos

Aplicar las mismas reglas para los videos.

→ Mostrar código 

  • <div class="flxVideo">
    <iframe frameborder="0" width="560" height="315" src="http://www.youtube.com/embed/2JI6h66NjcU?enablejsapi=0&autohide=1&theme=light&hd=1&modestbranding=1&rel=0&showinfo=0&showsearch=0&wmode=transparent"></iframe>
</div>

Sprites

Los ficheros editables de los sprites (en formato PSD de Photoshop) están en Dropbox (Dropbox > Fancybox > Sprites_PSD), por si hay que hacer cambios o crear versiones nuevas de los mismos. Disponemos de distintos tipos de sprites, en función de su naturaleza:

General Sprite

Para el General Sprite usaremos el tag img con una imagen transparente (inv.gif), junto con la class "s" seguida de la class correspondiente al sprite en concreto que queramos usar, p.e. "socFacebook".

→ Mostrar código 

<img src="img/inv.gif" width="15" height="15" class="s socFacebook" alt="Facebook" />

<img src="img/inv.gif" width="60" height="75" class="sWorldmap flagAr" alt="Argentina" />
